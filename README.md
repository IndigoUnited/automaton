Automaton website

---

Grunt must be installed.

# Development #
- Run `npm install`
- Run `grunt run`
- Open `index_dev.html`

# Updating doc #
- Run `grunt doc`

# Bulding #
- Run `npm install`
- Run `grunt build`

# Deploy #
Copy `img`, `dist`, `tmpl`, `index.html`, `.htaccess`, `.favicon` to the web root folder.
